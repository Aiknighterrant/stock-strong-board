# 更新日志

## v1.1.0 (2026-04-05)

### 🚀 主要优化

#### 1. 数据源系统重构
- **多数据源支持**：集成新浪财经、东方财富、akshare多个数据源
- **智能数据源选择**：自动选择可用的数据源，按优先级尝试
- **数据源降级机制**：当主要数据源失败时，自动切换到备用数据源
- **数据增强功能**：自动补充缺失的封单和板块数据

#### 2. 筛选逻辑优化
- **封单数据缺失处理**：当封单数据不可用时，使用成交额和换手率进行综合评分
- **优化异常值检测**：改进的IQR异常值检测算法
- **综合评分系统**：引入基于成交额、换手率、板块表现的综合评分
- **板块筛选改进**：更准确的板块强度评估算法

#### 3. 用户体验提升
- **数据源状态显示**：实时显示当前使用的数据源和获取状态
- **详细筛选过程**：每一步筛选都有详细的日志输出
- **错误友好提示**：提供清晰的错误信息和解决方案
- **缓存机制**：5分钟缓存，减少API调用，提高响应速度

#### 4. 技术架构改进
- **模块化设计**：数据获取、筛选、分析模块完全分离
- **错误恢复机制**：完善的异常处理和恢复流程
- **性能监控**：内置性能监控和日志系统
- **向后兼容**：保持与旧版本的兼容性

#### 5. 新增功能
- **实时数据验证**：自动验证数据完整性和准确性
- **数据质量报告**：显示数据获取的成功率和质量
- **自定义参数**：支持更多自定义筛选参数
- **多种输出格式**：支持表格、JSON、CSV多种输出格式
