# 沪深A股短线强势封板股分析技能

## 概述

本技能用于识别沪深A股市场中具有短线强势特征的封板股票。通过四步筛选流程，从当日涨停股池中找出成交活跃、封板强度高、换手率适中、所属板块表现良好的优质封板股。

## 工作流程

### 1. 获取涨停股池
根据用户提供的交易日期（默认为今日），自动获取当日所有涨停的A股股票列表。包含以下数据字段：
- 股票代码
- 股票名称
- 成交额（亿元）
- 封单金额（万元）
- 换手率（%）
- 所属板块

### 2. 筛选封板强度
对涨停股池中的每只股票按以下规则进行过滤：
1. 剔除当日成交额 ≤ 1亿元的股票
2. 排除封单金额异常高或异常低的极端值（使用IQR方法）
3. 计算每只股票的封板成交比 = 封单金额 / 成交额
4. 保留封板成交比最高的前25只股票

### 3. 过滤换手率
从上述25只股票中，剔除换手率 < 10% 或 > 20% 的个股。

### 4. 评估板块强度
分析剩余股票所属行业或概念板块在当日的整体表现：
1. 计算所属板块的当日涨跌幅
2. 计算所属板块的当日成交额占比
3. 剔除所属板块当日整体下跌的股票
4. 剔除所属板块活跃度显著低于市场平均水平的股票

### 5. 输出结果
返回符合条件的股票列表，每项包含以下字段：
- 股票代码
- 股票名称
- 成交额（单位：亿元）
- 封单金额（单位：万元）
- 封板成交比（保留三位小数）
- 换手率（%）
- 所属板块
- 板块当日涨跌幅（%）
- 筛选得分（综合评分）

## 快速开始

使用以下命令分析今日强势封板股：

```bash
python scripts/main.py --date today
```

或分析指定日期：

```bash
python scripts/main.py --date 2025-03-28
```

## 数据源配置

本技能使用以下免费数据源：
1. **东方财富网** - 获取涨停股池和基础数据
2. **新浪财经** - 获取封单金额和换手率数据
3. **同花顺** - 获取板块分类和表现数据

具体数据接口说明请参考 [references/data_sources.md](references/data_sources.md)。

## 参数说明

### 主要参数
- `--date`: 分析日期，格式为 YYYY-MM-DD 或 "today"（默认）
- `--output`: 输出格式，支持 "table"（表格）、"json"、"csv"
- `--limit`: 输出股票数量限制（默认10）

### 筛选参数
- `--min_turnover`: 最小换手率（默认10%）
- `--max_turnover`: 最大换手率（默认20%）
- `--min_amount`: 最小成交额（亿元，默认1）
- `--sector_threshold`: 板块涨跌幅阈值（默认-0.5%）

## 使用示例

### 示例1：分析今日强势封板股
```bash
python scripts/main.py --date today --output table
```

### 示例2：分析历史日期并保存为CSV
```bash
python scripts/main.py --date 2025-03-27 --output csv --output_file result.csv
```

### 示例3：自定义筛选参数
```bash
python scripts/main.py --date today --min_turnover 8 --max_turnover 25 --min_amount 0.5
```

## 注意事项

1. **数据延迟**：免费数据源可能存在15-30分钟的数据延迟
2. **交易日历**：自动跳过非交易日（周末、节假日）
3. **数据完整性**：部分股票可能因数据源限制而缺失某些字段
4. **结果仅供参考**：投资有风险，决策需谨慎
